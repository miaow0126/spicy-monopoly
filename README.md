# 涩涩大富翁 🎲🔥

> 一个**双人棋盘性爱游戏**。你把脚本和手册甩给一个能跑代码的 AI（ChatGPT 代码解释器 / Claude / Claude Code 等）——它**既是荷官、也是你的玩伴**：掷骰、走格、抽任务、算账都由代码算好真值，而 AI 不只是发牌的——**轮到它的任务，它自己真的演**，跟你来回互动、一起玩。（也可以两个真人玩、让 AI 只当荷官。）

---

## ⚠️ 成人内容 · 仅限 18+

- 本项目含**露骨的成人／性内容**，仅供**年满 18 岁（或你所在地成年年龄）的成年人**使用。

- 内置**安全词、红线过滤、跳过／换任务**机制（见下「安全设计」）——请务必先读、并在开局就跟对方说清楚。

## 这是什么

- **双人**轮流掷骰，绕 20 格棋盘走，踩到不同格子做不同的亲密任务。两个玩家可以是**真人 + Ta 的 AI**，也可以是两个真人——是 AI 的那个玩家既跑引擎当荷官，也**真的下场玩**、自己那份任务自己演。
- **金币经济**驱动：做任务赚币、占地盘收过路费，打满回合数**金币多的赢**，赢家可让输家执行一道「终极指令」。
- **不只是任务卡**：身份卡（每人一个贯穿全局的人设）、🌀淫纹（藏一处让对方猜）、⚔️同格对决、🎴功能卡、机会／未知格……一整套让每局都不一样。
- **包容任何组合**：AI／真人、性别、攻／受三者独立，男女／男男／女女局都支持。
- **任务库 900+ 张**，全部是**占位符写法**（「行动方 / 对方」），开局渲染成你们自己的名字。

## 三种玩法

### ⭐ 最省事 —— 直接调公开托管实例（零安装、零下载）
不想装、也不想传文件？把这个地址给你的 AI，让它照 `monopoly-API使用手册.md` 直接调：
```
https://spicy-monopoly.lol
```
（一个公开托管实例：本地库抽签、零 key、有限流。把手册里的接入地址换成它即可。请文明使用、别刷爆它 🙏）

### 或者以下玩法

### ① 用源码 —— 甩给你的 AI
把这几个文件交给你的 AI，让它**既当荷官、又当你的玩伴**，用**代码解释器**跑：
- `monopoly_play.py`（引擎，**零第三方依赖**，纯 Python 标准库）
- `monopoly-library.v2.json`（任务库）+ `monopoly-truths.json` + `monopoly-identities.json`
- `monopoly-给AI的操作手册.md`（喂给 AI 照着调）

AI 每回合就敲一条命令，不用你写代码、不碰文件：
```bash
python monopoly_play.py new "Alice:男:攻" "Bob:女:受" heavy 0.5 18   # 开局
python monopoly_play.py roll                                          # 每回合就这条
python monopoly_play.py status                                        # 看棋盘和账
```

### ② 自建 API 服务
想让 AI 通过 HTTP 调、或多桌同时玩：
```bash
pip install -r requirements.txt
uvicorn monopoly_api:app --host 0.0.0.0 --port 8069
```
接口用法见 `monopoly-API使用手册.md`。（公开对外时记得自己加鉴权和限流。）

## 安全设计（务必先读）

- **安全词 `404`**：任何人、任何时候说「404」，游戏立刻停，不追问理由。
- **红线**：开局把双方都不想碰的花样（后庭／打／绑／玩具／暴露／羞辱／失禁／电……）设进 `redline`，引擎**全程自动避开**，漏标的卡还有内容兜底二次拦。
- **后庭默认关**：不显式开，两人都不会抽到肛交任务；要玩得逐人 `open_anal` 打开。
- **纯 top**：`no_penetration` 里的人任何孔都不被插。
- **随时跳过 / 换任务**：不想做的任务一句话跳过，或花点代价换一道。

## 文件说明

| 文件 | 作用 |
|---|---|
| `monopoly_play.py` | 游戏引擎（掷骰／抽卡／渲染／红线／存档，零依赖） |
| `monopoly_api.py` | 可选的 HTTP 服务（FastAPI） |
| `monopoly-library.v2.json` | 任务库（占位符 · 四轴 schema） |
| `monopoly-truths.json` | 真心话库 |
| `monopoly-identities.json` | 身份卡库 |
| `monopoly-给AI的操作手册.md` | 给 AI（荷官兼玩家）照着调的说明书 |
| `monopoly-API使用手册.md` | HTTP 接口手册 |
| `monopoly-怎么玩-人类版.md` | 给人看的玩法简介 |

## 想改／扩库？

任务库是纯 JSON，字段直白（强度 / 玩法类型 / kink / 攻受需求 / 占位符内容），可以直接手改、增删。想成规模地生成新语料的工具链暂未随核心版发布。
