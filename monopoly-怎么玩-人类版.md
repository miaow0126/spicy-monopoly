# 涩涩大富翁 · 玩法说明(人类版)

## 这是什么
你和伴侣的棋盘性爱游戏。掷骰走格、踩格做任务、占地盘抢金币,最后**赢家通吃**、砸一道「不能拒绝」的终极指令收尾。

## 怎么开始玩(三步)
1. 把这三个文件发给一个**能跑代码的 AI**(ChatGPT 代码解释器、Claude 等):
   - `monopoly_play.py` —— 游戏引擎
   - `monopoly-library.v2.json` —— 任务库(放同目录)
   - `monopoly-给AI的操作手册.md` —— 告诉 AI 怎么跑
2. 跟它说:**「你当荷官,按操作手册跑这个游戏,我和 XX 玩。我们 [谁攻谁受],红线是 [不要的项]。」**
3. 它会掷骰、报任务、记账,你们照着做。**它只把任务发给你们、不替你俩描写你们那部分**(要是你就是在跟 AI 一起玩,那它踩到自己的任务也会真的下场演)。

## 棋盘怎么走
- 20 格环形棋盘,两人**轮流**掷骰前进。
- 玩满**每人 12 次掷骰**(也就是 12 个来回)游戏结束。
- 前半场前戏强度、后半场一起进高潮(两人同步升温)。

## 格子类型
- 🎯 **任务**:做任务,按强度给币(越敢做赚越多:轻 1 / 中 2 / 狠 3)。做完这格**归你**(占地盘)。
- 🚩 **地盘**:对方踩到你占的格子,要么**交过路费**给你、要么**听你差遣**(没钱只能受差遣 = 用身体抵)。
- 🛒 **商店**:花 3 币摸一张功能卡。
- 🎴 **抽卡**:摸功能卡(回退 / 抢劫 / 入狱 / 出狱 / 加速 / 赌一把 / 收租 / 敲诈)。
- ❓ **未知格**:好运(捡钱 / 发财 / 摸卡)或坏运(罚款 / 进监狱 / 超级任务)。
- 💬 **真心话**:答一个露骨问题。
- 🔒 **监狱**:被关 1 轮,轮到你时被绑、任对方处置,不能反抗。
- ⚔️ **同格对决**:两人走到同一格,对撩比谁先破功,**输家给赢家钱**。

## 金币能干嘛
攒钱 = 攒**收拾对方的本钱**。摸卡、被抢、交过路费都用币。**终局比谁钱多,赢家免费砸一道对方不能拒绝的终极指令**(除红线 / 404)。

## 安全
- 开局告诉荷官你的**红线**(不要的花样,共 13 类可选:后庭 / 打 / 绑 / 玩具 / 暴露 / 羞辱 / 失禁 / 足 / 口水 / 产乳 / 电 / 双龙 / 催眠 / 蜡),它全程避开。**后庭默认就是关的**,要玩才开。
- 任何时候喊 **"404" 立刻全停**。
- 疼痛、强度上限你自己定,开局告诉荷官。

## 玩不下去了?
如果 AI 卡住、报错、或开始**自己瞎编**——让它**重新读操作手册、从引擎重新开局**,别将就着玩它脑补出来的剧情(那已经不是真游戏了)。

## 反馈(完全可选)
玩得开心最重要。**游戏默认不会主动问你反馈,也绝不会打断你们的兴致。** 要是你们自己遇到了 bug、或有哪张任务卡别扭想吐槽,随时可以跟荷官 AI 说一声、或去项目的 GitHub Issues 提——这**完全不是必须的**,只是制作者想把游戏打磨得更好的一个小心愿。
